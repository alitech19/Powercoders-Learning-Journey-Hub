## Student dashboard {#overview}

Your home screen summarises learning activity: shortcuts to apps, task counts, and reminders (for example a due weekly reflection).

- **Workflows** (large card) — programmes you are enrolled in.
- **Goals · Tasks · Reflections** — quick stats and links.
- **Journal · Habits · Group · Resources** — same order as the top navigation.

Use **New Task** to create a personal task. Task tables at the bottom list Individual, Group, and Cohort assignments separately.

## Teacher dashboard {#overview-teacher}

You see missing weekly reflections for students in your groups, group activity, and **Group Public Tasks** you can supervise.

- Open a group card to create a **group task** (shared progress for the whole group).
- You do **not** see students' private personal tasks — only shared and group-visible work.

## Admin dashboard {#overview-admin}

<!-- role: admin -->

System-wide counts (students, teachers, cohorts, groups) and links to Django admin for user and cohort management.

- **All Public Tasks** — tasks visible as shared across the platform.
- **Private Tasks** — metadata only (title hidden); for audit, not content inspection.
