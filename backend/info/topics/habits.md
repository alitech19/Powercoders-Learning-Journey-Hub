## Habits list {#list}

Each habit has a weekly logging grid and streak counter. Mark individual days done or not done.

## Habit detail {#detail}

Week grid, streak info, and settings.

## Create habit {#form-create}

### Title (required)

Name of the habit (e.g. “Practice typing 30 min”).

### Description

Optional reminder why this habit matters to your learning.

### Target days per week

Number 1–7 — how many days you **aim** to do the habit. The grid compares logs against this target; it does not block you from logging on other days.

### Target minutes (optional)

If set, a time goal per session (informational; logging is still done/not done per day on the detail page).

### Visibility

- **Only me** — private habit and logs.
- **Share with teachers** — teachers can see the habit and your weekly grid.

Logging **done / not done** happens on the **detail** page after you create the habit, not on this form.

## Edit habit {#form-edit}

Change title, description, targets, visibility. Completed habits use **Complete** / **Reactivate** flows instead.

## Delete / complete habit {#form-delete}

**Complete** archives the habit. **Delete** removes it and logs.

## For students {#for-students}

Log each day on the detail page; dashboard counts how many “done” logs you made this week.
